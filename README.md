# Quiz Application using Java

## Project Description
This project is a console-based Quiz Application developed using Java.
It presents multiple-choice questions to the user, accepts answers,
calculates the score, and displays the final result.

## Objectives
- To understand basic Java programming concepts
- To use conditional statements and loops
- To handle user input using Scanner class

## Tools & Technologies Used
- Java (JDK)
- VS Code
- GitHub

## Features
- Multiple-choice questions
- User input validation
- Automatic score calculation
- Result display at the end of quiz

## Algorithm
1. Initialize score to zero
2. Display a question with options
3. Accept user answer
4. Compare with correct answer
5. Increase score if answer is correct
6. Repeat for all questions
7. Display final score
